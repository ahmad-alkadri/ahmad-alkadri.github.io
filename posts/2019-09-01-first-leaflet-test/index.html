First Leaflet Test

Posted at — Sep 1, 2019

This is my first blog post featuring a leaflet map. I’ve followed the step-by-step guide provided in this blog post and apparently it works.

To say that I’m happy would be an understatement. Now I’m able to write blog post with maps embedded in it. I’m going to look up how to “beautify” this website though; perhaps I’m going to find some custom themes. God knows how bad I am with CSS.

Anyway, here’s the birthplace of R language.

library(leaflet)

m <- leaflet() %>%
  addTiles() %>%  # Add default OpenStreetMap map tiles
  addMarkers(lng=174.768, lat=-36.852, 
  popup="The birthplace of R")
m  # Print the map
